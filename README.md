HXPictureTextMixingView是一款图文混排的View

特性： 

-图片在文字前面 

-图片在文字后面

-支持单行 多行显示 

-支持高度计算方法+ (CGFloat)getHeightWithModel:(HXPictureTextMixingViewModel *)model;
